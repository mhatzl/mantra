# mantra-schema

[![crates.io](https://img.shields.io/crates/v/mantra-schema.svg)](https://crates.io/crates/mantra-schema)

This crate contains Rust structs that define the JSON schemas for [mantra](https://crates.io/crates/mantra).
Generated JSON schemas are available in the mantra repo at [/schema-gen/generated](https://github.com/mhatzl/mantra/tree/main/schema-gen/generated).
